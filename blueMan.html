Blue Man Group, New Interactive Percussion Set

The classic Blue Man Group's Percussion set has been reworked to be fully interactive with Blue Man's performance.


Interative visuals are displayed through 118*92 LEDs, are fully reactive to successive drum hits. The resulting visuals are based on real-time audio FFT analysis.

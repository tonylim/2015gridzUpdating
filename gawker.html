PORTFOLIO

Gawker Media Tech Party, Leap Motion Controller based audio visual installation

Audio visualization installation commissioned by Gawker.com


The installation features Leap Motion Controller as the key interaction device. The resulting visual is based on real-time audio FFT analysis. Based on the finger gestures, each finger will transform a designated element of the visual, which is simultaneously being transformed by the current music being played by the DJ.
